4. Gaussian Kernel (let D be infinite dimension)

We can inspect what the $\Phi(x)$ is, but we don‚Äôt use it in practice

üí° It‚Äôs best ******not****** to think of points in a high-d space. It‚Äôs no longer a useful intuition. Instead, think of the kernel $k$ as a measure of how similar or close together two points

Referencing back to the dual ridge regression, now the $h(z)$ in test time is a linear combination of Gaussians centered at x. $h(z)=\sum_{j=1}^na_j\cdot k(X_j,z)$ ‚Üê dual weights are the coefficients

  • Intuition from Gaussian kernel function (a measure of similarity)

The Gaussian kernel is very popular in practice

üí° Note: we treat $\sigma$ in kernel function as a hyper-parameter. Larger $\sigma$ ‚Üí wider Gaussians & smoother $h$ ‚Üí more bias & less variance.
